• Block Reward:     0.01 ACG (Gold bar)

• Difficulty:       Every block

• Halving:          None

• Ticker:           ACG

• Block size:       4mb

• TPS:      	    9.52 (Transaction per second)

• P2P Port:         9889 - 19889 - 29889

• RPC Port:         9888 - 19888 - 29888

• Max supply:       18,999,999 ACG

• POW Maturity:     100 blocks
